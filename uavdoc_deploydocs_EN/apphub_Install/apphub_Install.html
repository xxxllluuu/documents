3. AppHub Installation

AppHub Installation

AppHub is a JEE application that displays data and controls management functions. It is deployed on JEE servers (such as Tomcat) in the format of war package.

Installation and Configuration

Modify the configuration of web.xml in the war package >< param-name >uav.server.port< /param-name >< param-value>AppHubStartPort< /param-value >

< param-name >uav.app.gui.redis.store.addr< /param-name > < param-value>guiCacheAddress(Redis)< /param-value >

< param-name >uav.app.manage.redis.store.addr< /param-name > < param-value>appManagementInformationCacheAddress(Redis)< /param-value >

< param-name >uav.app.ldap.redis.store.addr< /param-name > < param-value>accessManagementInformationCacheAddress(Redis)< /param-value >

< param-name >uav.app.godeye.redis.store.addr< /param-name > < param-value>monitoringDataInformationCacheAddress(Redis)< /param-value >

< param-name >@url:uav.app.godeye.hbquery.http.addr< /param-name > < param-value>serverAddressForHeartbeatDataQueryInAppHubManager< /param-value >

< param-name >@url:uav.app.godeye.healthmanager.http.addr< /param-name > < param-value>serverAddressForMonitoringDataQueryInAppHubManager< /param-value >

< param-name >@url:uav.app.godeye.notify.view.http.addr< /param-name > < param-value>notificationServerAddressInProfileAndNotificationServer< /param-value >

< param-name >@url:uav.app.godeye.notify.strategy.http.addr< /param-name > < param-value>runtimeNotificationServerAddressInAppHubManager< /param-value >

< param-name >@url:uav.app.manage.apphubmanager.http.addr< /param-name >
< param-value>AppHubManagerAddressInAppHubManager< /param-value >

< param-name >@url:uav.app.upgrade.server.http.addr< /param-name > < param-value>serverAddressInUpgradeServer< /param-value >

< param-name >@url:uav.app.apm.ivc.http.addr< /param-name > < param-value>invocationChainAddressInApplicationPerformanceManager< /param-value >

< param-name >@url:uav.app.apm.jta.http.addr< /param-name > < param-value>threadAnalysisAddressInApplicationPerformanceManager< /param-value >

< param-name >@url:uav.app.hm.newlog.http.addr< /param-name > < param-value>loggingAddressInLog Server< /param-value >

< param-name >uav.apphub.sso.ldap.connection.info< /param-name > < param-value>LDAPconfigurationInformation< /param-value >

< param-name >uav.apphub.sso.grafana.connection.info< /param-name > < param-value>GrafanaconfigurationInformation< /param-value >

Copy the war package into the application directory of JEE server, such as webapps directory of Tomcat.

Start and Login

Start the JEE server (such as sh bin/start.sh under Tomcat), access http://ip:port/apphub and sign in AppHub with the account admin and the password admin.
