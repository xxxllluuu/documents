2. AppHub

AppHub Configuration

Configure AppHub in contextparam of web.xml. Below is the illustration of parameters

uav.app.*.redis.store.params

Configuration parameters of Redis connections for cache

uav.app.*.http.client.params

Configuration parameters of Http connections

uav.app.godeye.notify.mongodb.query.range

Optimized parameters for Mongo query

uav.apphub.sso.admin.loginid

Admin username

uav.apphub.sso.admin.password

Admin password

uav.apphub.sso.guest.loginid

Guest username

uav.apphub.sso.guest.password

Guest password

uav.apphub.sso.white.list

Configuration of white list users

uav.app.manage.redis.store.addr

Address (Redis) of App management information cache

uav.app.ldap.redis.store.addr

Address (Redis) of access management information cache

uav.app.godeye.redis.store.addr

Address (Redis) of monitoring data cache

@url:uav.app.godeye.hbquery.http.addr

Address of heartbeat data query server in AppHub Manager

@url:uav.app.godeye.healthmanager.http.addr

Address of monitoring data query server in AppHub Manager

@url:uav.app.godeye.notify.view.http.addr

Address of notification server in Profile and Notification Server

@url:uav.app.godeye.notify.strategy.http.addr

Address of runtime nofitication server in AppHub Manager

@url:uav.app.manage.apphubmanager.http.addr

Address of AppHub Manager in AppHub Manager

@url:uav.app.upgrade.server.http.addr

Address of Upgrade Server

@url:uav.app.apm.ivc.http.addr

Address of invocation chain server in Application Performance Manager

@url:uav.app.apm.jta.http.addr

Address of the thread analysis server in [Application Performance Manager

@url:uav.app.hm.newlog.http.addr

Address of Log Server

uav.apphub.sso.implclass

Full class name of implementation class for authority verification

uav.apphub.sso.ldap.connection.info

LDAP configuration information

uav.apphub.sso.grafana.connection.info

Grafana configuration information
