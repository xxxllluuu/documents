Upgrade Server (upgrade_server)

Upgrade Server

Configuration file upgrade_pro(test)_server.properties

Log Configuration

log.level=DEBUG

INFO, ERR, WARNING, DEBUG, FINE, FINER and ALL

log.debug=true

Enable debug logging or not

log.fsize=10

Size of single output block of logs

meta.datacenterid=1

Configure data center ID

meta.nodetype=\u5347\u7ea7\u4e2d\u5fc3

Node name and UTF8 code

Heartbeat Client Feature

feature.hbclientagent.loader=com.creditease.uav.agent.heartbeat.jar

Main jar of the heartbeat client feature for classloader isolation

feature.hbclientagent.enable=true

Enable the heartbeat client feature

feature.hbclientagent.components=com.creditease.agent.feature.HeartBeatClientAgent

Main class of the heartbeat client feature

feature.hbclientagent.properties=

Property configuration, for example: key1=value1,key2=value2,...

feature.hbclientagent.hbhandlers=com.creditease.agent.feature.hbagent.handlers.HBClientDefaultHandler

Handler class of heartbeat delivery

feature.hbclientagent.hbservers=127.0.0.1:8010

Heartbeat server address

feature.hbclientagent.interval=15000

Heartbeat delivery interval, in ms

Node Control Feature

feature.nodeoperagent.loader=com.creditease.uav.agent.heartbeat.jar

Main jar of the node control feature for classloader isolation

feature.nodeoperagent.enable=true

Enable the node control feature

feature.nodeoperagent.components=com.creditease.agent.feature.NodeOperAgent

Main class of the node control feature

feature.nodeoperagent.http.port=10500

Port of node control server

Upgrade Server Configuration

feature.upgradeserver.loader=default

Load the node control feature with the default class loader

feature.upgradeserver.enable=true

Enable the upgrade server or not

feature.upgradeserver.components=com.creditease.uav.feature.upgrade.download.UpgradeDownloadServer

Main class of the upgrade server feature

feature.upgradeserver.drhandlers=com.creditease.uav.feature.upgrade.download.UpgradeDownloadRequestHandler

Handler class of upgraded download requests of the upgrade server feature

feature.upgradeserver.http.port=8888

Port of the upgrade server

feature.upgradeserver.http.backlog=10

Length of the Http connection queue of the upgrade server

feature.upgradeserver.http.bossgroup.size=2

Thread count received by Http of the upgrade server

feature.upgradeserver.http.wokergroup.size=10

Thread count handled by Http of the upgrade server

feature.upgradeserver.download.dir=D:/packages

Directory of the installation package of the upgrade server

feature.upgradeserver.download.threshold=10

Maximum simultaneous downloads

Work queue configuration of the upgrade server

feature.upgradeserver.1nqworker.coresize=10

1:N core worker queue count of the upgrade server

feature.upgradeserver.1nqworker.maxsize=10

1:N maximum worker queue count of the upgrade server

feature.upgradeserver.1nqworker.bqsize=1

Length of 1:N worker queue of the upgrade server

feature.upgradeserver.1nqworker.keepalivetimeout=60000

Survival period of 1:N worker queue of the upgrade server
